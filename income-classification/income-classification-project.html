Abstract¶

This notebook evaluates and compares multiple classification models to examine how regularization and ensemble methods affect predictive performance. The analysis begins with Linear Probability Models and Logistic Regression, then introduces regularization techniques including Lasso, Ridge, and Elastic Net to control model complexity and improve stability. Advanced tree-based models, specifically Random Forest and Gradient Boosting, are then applied to capture nonlinear patterns and interactions in the data. Model performance is assessed using confusion matrices, classification metrics, feature importance analysis, and error rate comparisons. Results show that ensemble methods, particularly Gradient Boosting, achieve the strongest balance of accuracy, recall, and error reduction. The notebook concludes with a model recommendation supported by empirical results, highlighting the trade-offs between predictive performance, interpretability, and practical deployment considerations.

Model 2: Linear Probability Model with Lasso Regularization¶

  • This model applies Lasso (L1) regularization to the Linear Probability Model. Lasso adds a penalty to the absolute magnitude of coefficients, which can shrink some coefficients to zero and perform implicit feature selection.
  • The regularization strength (alpha) is tuned using cross-validation to identify the optimal penalty that balances bias and variance.

# Metric Function
def compute_metrics(y_true, y_pred):

    tn, fp, fn, tp = confusion_matrix(y_true, y_pred).ravel()

    accuracy = (tp + tn) / (tp + tn + fp + fn)
    precision = tp / (tp + fp)
    recall = tp / (tp + fn)
    
    specificity = tn / (tn + fp)
    
    f1 = 2 * (precision * recall) / (precision + recall)

    type1 = fp / (fp + tn)
    type2 = fn / (fn + tp)

    return accuracy, precision, recall, f1, specificity, type1, type2

5. Model Recommendation and Justification

This section identifies the best model based on empirical results. It explains the final model recommendation, evaluates trade-offs between performance and interpretability, and reflects on insights from regularization and ensemble methods.

a) Recommended Model¶

  • Top Recommended Model: Gradient Boosting is the best model. It achieves the highest accuracy (0.8778), highest precision (0.8153), highest recall (0.6585), and highest F1-score (0.7285). It also has the lowest Type II error (0.3415) and a low Type I error (0.0494).

  • Second-Best Alternative: Random Forest is a strong backup. It performs slightly lower than Gradient Boosting but remains stable with high accuracy (0.8696) and strong F1-score (0.7080).

  • Performance vs Interpretability: Gradient Boosting provides the best predictive performance, but it is less interpretable than linear models like Logistic Regression. However, the performance gains justify the reduced interpretability for prediction tasks.

  • Error Trade-off in Income Prediction: In income prediction, Type II errors (missing high-income individuals) are often more costly because they hide important signals about high earners. Gradient Boosting aligns with this goal by achieving the lowest Type II error rate (0.3415) among all models.

b) Reflection¶

  • Role of Regularization: Regularization helped stabilize the linear models by shrinking coefficients and reducing overfitting risk. However, it did not significantly improve overall performance for this dataset. Ridge slightly improved Logistic Regression, while Lasso and Elastic Net often reduced recall and overall accuracy. This suggests the dataset benefits more from keeping most predictors rather than aggressively removing them.

  • Random Forest vs Gradient Boosting: Gradient Boosting would be preferred for deployment because it achieved the highest accuracy (0.8778) and F1-score (0.7285) while maintaining strong precision and recall. Random Forest is still valuable because it trains faster and is easier to tune. However, when predictive performance is the main priority, Gradient Boosting provides better results.

  • Explaining the Model to Non-Technical Stakeholders: The model works by learning patterns from past data to predict whether someone earns a high income. It analyzes many factors such as age, education, work hours, and capital gains to identify patterns linked to higher earnings. By combining many decision trees, the model improves accuracy and reduces mistakes compared with simpler models. We trust the model because it consistently performs best across multiple evaluation metrics and handles complex relationships in the data effectively.
